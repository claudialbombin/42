## Description

Growing Code is a small Python exercise set focused on writing simple functions, using basic control flow, and getting comfortable with recursive and iterative thinking.

The project is organized as a collection of short exercises, each one in its own folder. A helper script, `main.py`, is included to make it easy to test the exercises from the command line.

The exercises in this repository are:

- `ft_hello_garden`
- `ft_garden_name`
- `ft_plot_area`
- `ft_harvest_total`
- `ft_plant_age`
- `ft_water_reminder`
- `ft_count_harvest_iterative`
- `ft_count_harvest_recursive`
- `ft_seed_inventory`

## Instructions

### Run the helper script

```bash
python3 main.py
```

This opens a small menu that lets you test each exercise individually or run them all.

### Exercise files

Each exercise lives in its own file inside its corresponding `ex*` folder. The function name and the file name match, so the helper script can import them directly.

### Example

If you want to test one exercise manually, you can run it directly with Python:

```bash
python3 ex0/ft_hello_garden.py
```

## Project Overview

The project covers basic Python topics through very small tasks:

- printing messages
- reading user input
- simple arithmetic
- basic conditionals
- loops
- recursion
- type hints in the final exercise

`ft_seed_inventory` is the only exercise that expects parameters. The helper script shows how to call it with different values.
